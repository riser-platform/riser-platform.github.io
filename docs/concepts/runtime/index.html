        Environment Variables #

        The Riser runtime injects environment variables that provide information about the runtime or your app itself.

        NameDescription
        RISER_APPThe name of the Riser app
        RISER_DEPLOYMENTThe name of the Riser deployment
        RISER_DEPLOYMENT_REVISIONThe revision number of the Riser deployment
        RISER_ENVIRONMENTThe name of the Riser environment
        RISER_NAMESPACEThe name of the Riser namespace
